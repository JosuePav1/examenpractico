# 🚩 Buscaminas - Programación Orientada a Objetos

## 📋 Descripción del Proyecto
Implementación del clásico juego **Buscaminas** en consola desarrollado en **Java** utilizando el paradigma de **Programación Orientada a Objetos (POO)**.

El juego cuenta con un tablero de 10x10 con 10 minas, sistema de revelado en cadena, marcación de banderas, manejo de excepciones, patrón MVC y buenas prácticas de programación.

## ✨ Características Implementadas

- Tablero de 10x10 con 10 minas colocadas aleatoriamente
- Revelado en cadena (flood fill) al descubrir casillas vacías
- Sistema de banderas (🚩) para marcar minas sospechosas
- Uso de emojis para mejor experiencia visual (💣 🚩 ■)
- Manejo de excepciones personalizadas
- Patrón **MVC** (Modelo - Vista - Controlador)
- Revelado de **todas las minas** al perder
- Detección automática de victoria
- Código limpio y bien organizado en paquetes
- Pruebas unitarias con JUnit 5

---

## 🛠️ Tecnologías Utilizadas

- **Lenguaje:** Java
- **IDE:** Eclipse
- **Paradigma:** Programación Orientada a Objetos (POO)
- **Patrón de diseño:** MVC
- **Pruebas:** JUnit 5

---

## 🚀 Cómo Ejecutar el Juego

1. Abrir el proyecto en **Eclipse**
2. Asegurarse que la carpeta `src` esté configurada como Source Folder
3. Ejecutar la clase **`principal/Main.java`**
4. Seguir las instrucciones que aparecen en la consola

---

## 🎮 Cómo Jugar

- **`A5`** → Descubrir la casilla
- **`M A5`** → Marcar o desmarcar con bandera 🚩
- **`S`** → Salir del juego
